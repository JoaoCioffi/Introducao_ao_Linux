To run a shell script:

![image](https://user-images.githubusercontent.com/60454486/170837280-3463207e-e4b1-48b5-baaa-729eac0638c9.png)


1- Change reading permissions:
```
chmod R 777 .
```

2- Run the script (Kernel)
```
./script.sh
```
