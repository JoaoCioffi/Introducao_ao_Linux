```

```

2- Open with nano and edit content:
```
nano test.txt
```

## Via terminal

1- Visualize a file content:
```
cat test.txt
```

2- Visualize head and tail of a file (10 first and/or 10 last rows):
```
head test.txt

tail test.txt

tail test.txt > tail_content.txt
```
