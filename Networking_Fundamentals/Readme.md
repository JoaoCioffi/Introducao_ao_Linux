# Networking Fundamentals and Advanced Commands

## Networking 🌏

### 1- Read Networking Interfaces

```
sudo apt install net-tools
```

```
ifconfig
```

### 2- Host info
```
hostname
```

```
hostname -I #ip address
```

```
hostname -i #loopback ip address
```
